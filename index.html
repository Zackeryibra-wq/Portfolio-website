My Work

Here are some projects I’ve created to showcase my web design and development skills. The project section uses CSS Grid to ensure the layout works smoothly on all devices.

Food Delivery Website

This project is a responsive delivery website where users can browse restaurants, view menus, and place orders online. The focus was on creating a clean design, intuitive navigation, and mobile-first layout to ensure an excellent user experience.

Barber Appointment Booking Web App

Technologies: HTML5, CSS3, Flexbox, JavaScript, Responsive Design
